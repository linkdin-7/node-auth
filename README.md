Technologies Used
 Express (Server)
 Nodemon (Dev)
 Mongoose (DB/Schema)
 Morgan (Logging)
 Passport (Auth)
 bCrypt (Encrypt/Decrypt)
 JWT (Tokens)

How to run:
1. run npm start
2. Make change of secret value in config file as per your requirement.
3. run nodemon index.js
